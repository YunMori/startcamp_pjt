LocalHub - 연인 데이트 코스 추천 & 지역 커뮤니티 플랫폼의 프론트엔드를 만들어줘.
서비스의 모든 핵심 경험이 챗봇과의 대화 형식으로 진행되는 구조로 디자인해줘.

[프로젝트 개요]
- 서비스명: LocalHub
- 컨셉: 대전/충청권 연인들을 위한 맞춤형 데이트 코스 추천 + 익명 커뮤니티
- 타깃: 맞춤형 데이트 코스를 원하는 연인, 대전/충청권 방문객
- 기술스택: Vue.js 3 (SPA), Kakao Map API, 모바일 우선 반응형

[디자인 톤앤매너]
- 따뜻하고 감성적인 데이트 앱 느낌 (핑크/코랄 계열 포인트 컬러 + 뉴트럴 베이스)
- 챗봇이 대화 상대처럼 느껴지는 친근한 UI (말풍선, 부드러운 애니메이션)
- 모바일 화면 기준으로 우선 제작

[전체 UX 플로우 — 전부 챗봇 대화 안에서 진행]

1단계. 설문조사 (챗봇 대화형)
   - 앱 진입 시 챗봇이 화면 전체를 차지하는 대화창으로 시작
   - 챗봇이 먼저 인사 후, 한 번에 하나씩 질문을 던지며 설문 진행
     (예: "오늘 데이트, 실내가 좋아요 실외가 좋아요?", "예산대는 어느 정도로 생각하세요?", "분위기는 조용한 곳 vs 활기찬 곳?")
   - 각 질문마다 텍스트 입력 대신 선택 가능한 버튼(칩 형태)을 말풍선 안에 함께 노출해 빠르게 답할 수 있게
   - 답변할 때마다 대화 히스토리에 사용자 답변이 쌓이는 형태

2단계. 데이트 코스 3종 추천 (챗봇 응답 카드 + Kakao Map)
   - 설문 완료 후 챗봇이 "이런 코스는 어때요?" 메시지와 함께 추천 코스 3개를 카드 형태로 대화창 안에 인라인 노출
   - 각 카드: 코스 썸네일, 코스명, 태그(예: #조용한 #실내 #가성비), 소요시간
   - 카드 상단 또는 배경에 Kakao Map 기반 미니 지도 미리보기(핀 3개 정도)로 코스 위치 감 잡기
   - 카드를 탭하면 아래 3단계로 이어짐

3단계. 사이드 패널 확장 (지역/축제 소개 + 리뷰 CRUD)
   - 카드 선택 시 화면 오른쪽(모바일에서는 하단→전체 확장)에서 사이드 패널이 슬라이드 업/인 되며 열림
   - 패널 상단: 선택한 코스가 속한 지역 또는 축제 소개 (사진, 설명, 기간 등)
   - 패널 하단: 해당 지역/축제에 대한 익명 리뷰 게시판 (CRUD)
     · 리뷰 목록 (닉네임, 별점 또는 만족도, 작성일)
     · 리뷰 상세 보기
     · 리뷰 작성 (비밀번호 기반, 로그인 없음)
     · 리뷰 수정/삭제 (비밀번호 확인)
   - 패널은 챗봇 대화창을 완전히 덮지 않고 옆에서 열려, 챗봇과의 대화 맥락이 유지되는 느낌으로 (닫기 버튼으로 대화창 복귀)
   - 패널 내에서 "챗봇에게 다른 곳 물어보기" 버튼으로 다시 대화 흐름으로 복귀 가능

[요청사항]
- 1→2→3단계가 하나의 연속된 대화 세션 안에서 끊김 없이 이어지는 것으로 표현
- 실제 클릭 가능한 요소(설문 선택 버튼, 추천 카드, 사이드 패널 열기/닫기, 리뷰 작성 폼) 포함
- 사이드 패널이 열리고 닫히는 인터랙션을 목업에서 확인할 수 있게 표현
